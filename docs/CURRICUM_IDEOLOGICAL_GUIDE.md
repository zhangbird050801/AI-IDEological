# 基于AIGC的《软件工程》课程思政内容生成及管理系统使用指南

## 系统概述

本系统是一个深度整合课程体系、微课素材、虚拟仿真资源等多元教学内容的教学辅助平台，通过AIGC技术帮助教师快速生成高质量的《软件工程》课程思政教学案例，提升教学效果。

## 核心功能

### 1. 智能内容生成
- **AIGC对话生成**：支持流式和批量生成课程思政案例
- **提示词模板库**：提供专业的提示词模板，确保生成质量
- **多样化内容类型**：支持案例分析、讨论题、思考题等多种形式
- **智能优化建议**：对已有内容提供优化建议

### 2. 案例库管理
- **案例分类**：按软件工程章节、思政主题、案例类型分类
- **标签系统**：灵活的标签管理，支持多维度检索
- **评分系统**：用户评分和评价，形成优质内容推荐
- **批量操作**：支持批量导入、导出、状态更新

### 3. 提示词模板管理
- **系统模板**：预置专业提示词模板
- **自定义模板**：支持创建个性化模板
- **模板变量**：动态变量替换，提高模板灵活性
- **模板分类**：按用途和类型分类管理

### 4. 教学资源管理
- **文件上传**：支持文档、视频、音频、图片等多种格式
- **外部链接**：集成虚拟仿真资源和外部平台链接
- **资源预览**：在线预览图片、文档等资源
- **访问控制**：支持公开和私有资源设置

## 快速开始

### 环境要求
- Python 3.11+
- Node.js 18.8.0+
- pnpm (推荐)

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/your-repo/AI-IDEological.git
cd AI-IDEological
```

2. **后端环境配置**
```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Linux/Mac
source venv/bin/activate
# Windows
.\venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑.env文件，配置DeepSeek API密钥等
```

3. **前端环境配置**
```bash
cd web

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

4. **系统初始化**
```bash
# 在项目根目录运行
python scripts/init_system.py
```

5. **启动后端服务**
```bash
python run.py
```

6. **访问系统**
- 前端地址：http://localhost:3000
- 后端API文档：http://localhost:9999/docs
- 默认管理员账号：admin / 123456

## 详细使用指南

### AIGC内容生成

#### 基础对话生成
1. 进入"AIGC思政案例生成"页面
2. 在输入框中输入生成要求，例如：
   ```
   请生成一个关于软件测试的工匠精神案例，适合本科生使用
   ```
3. 点击发送，系统将实时生成内容
4. 可以保存生成结果为案例

#### 使用提示词模板
1. 进入"提示词模板库"页面
2. 浏览或搜索合适的模板
3. 点击"使用模板"或在模板页面填写变量
4. 系统将基于模板生成专业内容

#### 增强生成功能
- **章节定向**：指定软件工程章节
- **主题定制**：选择具体思政主题
- **类型选择**：选择案例、讨论题、思考题等
- **流式输出**：实时查看生成过程

### 案例库管理

#### 创建案例
1. 进入"案例库"页面
2. 点击"新建案例"
3. 填写案例信息：
   - 标题和内容
   - 软件工程章节
   - 思政主题
   - 案例类型
   - 难度等级
   - 关键知识点
   - 讨论问题
   - 教学建议
4. 设置公开性并保存

#### 案例检索
- **关键词搜索**：搜索标题或内容
- **筛选条件**：章节、主题、类型、难度
- **标签过滤**：按标签筛选相关案例
- **视图模式**：网格或列表视图

#### 案例评价
- **用户评分**：1-5星评分
- **使用统计**：记录使用次数
- **收藏功能**：收藏优质案例

### 提示词模板管理

#### 创建模板
1. 进入"提示词模板库"页面
2. 点击"新建模板"
3. 填写模板信息：
   - 模板名称和描述
   - 模板类型
   - 模板内容（使用{{变量名}}定义变量）
   - 分类和标签
4. 系统自动提取变量

#### 变量使用
- 在模板内容中使用`{{变量名}}`格式
- 支持中英文变量名
- 自动验证变量有效性

#### 模板类型
- **案例生成**：生成教学案例
- **讨论生成**：生成讨论题
- **思考生成**：生成思考题
- **内容优化**：优化现有内容
- **教学设计**：生成教学方案
- **知识点**：融入思政元素

### 教学资源管理

#### 文件上传
1. 进入"教学资源中心"页面
2. 点击"上传资源"
3. 选择文件并填写信息：
   - 资源标题
   - 描述和标签
   - 适用章节和主题
4. 设置公开性并上传

#### 外部链接
1. 点击"添加链接"
2. 填写链接信息：
   - 资源标题
   - 链接地址
   - 描述和分类
3. 保存并分享

#### 资源类型
- **文档**：PDF、Word、PPT等
- **视频**：MP4、AVI等格式
- **音频**：MP3、WAV等格式
- **图片**：JPG、PNG等格式
- **演示文稿**：PPT、Keynote等
- **虚拟仿真**：外部仿真平台链接

## 技术支持

### 常见问题

**Q: 如何配置DeepSeek API密钥？**
A: 在项目根目录的.env文件中添加：DEEPSEEK_API_KEY=your_api_key

**Q: 如何备份系统数据？**
A: 定期备份sqlite数据库文件和uploads目录

**Q: 如何自定义系统模板？**
A: 管理员可以登录后端管理系统，修改系统模板

**Q: 支持哪些文件格式？**
A: 支持常见文档、图片、音视频格式，详见资源类型说明

### 日志查看
- 后端日志：logs/app.log
- 错误日志：logs/error.log
- 系统日志：logs/system.log
